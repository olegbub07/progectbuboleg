Описание тестов для проекта "Менеджер заметок"
Тесты для моделей (test_models.py)
1. test_note_creation

Что проверяет: Создание новой заметки

    Проверяет, что заголовок и содержание устанавливаются правильно

    Проверяет, что ID изначально равен None

    Убеждается, что заметка создается с корректными данными

2. test_to_dict

Что проверяет: Преобразование заметки в словарь

    Проверяет, что метод to_dict() возвращает словарь с правильными данными

    Убеждается, что все поля (id, title, content, created_at) присутствуют

    Проверяет корректность значений в словаре

3. test_from_dict

Что проверяет: Создание заметки из словаря

    Проверяет, что метод from_dict() создает объект Note из словаря

    Убеждается, что все поля правильно восстанавливаются

    Проверяет корректность данных в созданном объекте

Тесты для хранилища (test_storage.py)
4. test_save_note

Что проверяет: Сохранение заметки в хранилище

    Проверяет, что заметка успешно сохраняется

    Убеждается, что сохраненной заметке присваивается ID

    Проверяет работу метода save_note()

5. test_get_all_notes

Что проверяет: Получение всех заметок

    Сохраняет заметку, затем получает все заметки

    Проверяет, что количество возвращенных заметок равно 1

    Убеждается, что метод возвращает корректный список

6. test_delete_note

Что проверяет: Удаление существующей заметки

    Сохраняет заметку, затем удаляет ее по ID

    Проверяет, что метод возвращает True при успешном удалении

    Убеждается, что заметка действительно удаляется

7. test_delete_nonexistent_note

Что проверяет: Удаление несуществующей заметки

    Пытается удалить заметку с несуществующим ID (999)

    Проверяет, что метод возвращает False

    Убеждается в корректной обработке отсутствующей заметки

8. test_search_notes

Что проверяет: Поиск заметок по тексту

    Сохраняет заметку с текстом "Python"

    Ищет заметки по запросу "Python"

    Проверяет, что находится одна заметка

    Убеждается в корректности работы поиска

Тесты для команд (test_commands.py)
9. test_add_note_success

Что проверяет: Успешное добавление заметки

    Имитирует добавление заметки с заголовком и содержанием

    Проверяет вывод сообщения об успешном добавлении

    Убеждается, что метод save_note вызывается

10. test_add_note_empty_title

Что проверяет: Добавление заметки с пустым заголовком

    Пытается добавить заметку с пустым заголовком

    Проверяет исключение ValueError с помощью assertRaises

    Убеждается, что save_note не вызывается

11. test_add_note_empty_content

Что проверяет: Добавление заметки с пустым содержанием

    Пытается добавить заметку с пустым содержанием

    Проверяет исключение ValueError с помощью assertRaises

    Убеждается, что save_note не вызывается

12. test_list_notes_empty

Что проверяет: Показ пустого списка заметок

    Имитирует ситуацию, когда заметок нет

    Проверяет вывод сообщения "Заметок пока нет"

    Убеждается в корректном поведении при отсутствии данных

13. test_list_notes_with_data

Что проверяет: Показ списка с заметками

    Имитирует наличие заметок в хранилище

    Проверяет вывод информации о заметках

    Убеждается, что данные отображаются корректно

14. test_search_notes_empty

Что проверяет: Поиск с пустым запросом

    Пытается выполнить поиск с пустой строкой

    Проверяет вывод сообщения "Введите текст"

    Убеждается, что search_notes не вызывается

15. test_delete_note_success

Что проверяет: Успешное удаление заметки

    Имитирует удаление существующей заметки

    Проверяет вывод сообщения об успешном удалении

    Убеждается, что delete_note вызывается с правильным ID

16. test_delete_note_not_found

Что проверяет: Удаление несуществующей заметки

    Имитирует попытку удаления несуществующей заметки

    Проверяет вывод сообщения "не найдена"

    Убеждается в корректной обработке отсутствующей заметки
